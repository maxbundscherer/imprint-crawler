# Legal Notice Crawler

A legal notice crawler...

``metadata - extractor - imprint - website - scala``

## Let's get started

### Required

- ``java``
- ``sbt``

### Let's go (developer)

- Run with ``sbt run``
- (or) Run with ``sbt ~run``
